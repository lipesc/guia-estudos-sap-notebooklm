# 🎯 Biblioteca de Prompts Reutilizáveis

## Para Revisões Futuras

Use estes prompts para revisar o conteúdo no NotebookLM.

---

## 📚 Prompts de Revisão Geral

### Revisão Rápida
```
Faça um resumo dos 5 conceitos mais importantes sobre Quality Engineering 
para SAP S/4HANA que estudamos nos documentos.
```

### Revisão por Tema
```
Revise os principais pontos sobre [TEMA ESPECÍFICO] mencionados nos documentos.
Organize em: conceito, aplicação prática, ferramentas.
```

---

## 🧠 Prompts de Auto-Avaliação

### Quiz
```
Crie 5 perguntas de múltipla escolha sobre [TEMA] para testar meu conhecimento.
Inclua as respostas e justificativas.
```

### Verificação de Compreensão
```
Liste 3 cenários práticos envolvendo [CONCEITO] e pergunte qual seria a 
melhor abordagem. Depois forneça a resposta correta com justificativa.
```

---

## 🔍 Prompts de Aprofundamento

### Explorar Gap
```
Explique mais detalhadamente sobre [CONCEITO QUE FICOU CONFUSO], 
incluindo exemplos práticos e referências aos documentos.
```

### Conexões
```
Como [CONCEITO A] se relaciona com [CONCEITO B]? 
Forneça exemplos de quando usar cada um no contexto SAP.
```

---

## 📋 Prompts de Aplicação

### Caso Prático
```
Cenário: [Descreva uma situação do trabalho]
Com base no que estudamos, como eu deveria abordar este problema?
```

### Checklist
```
Crie um checklist de verificação para implementar [PROCESSO/FERRAMENTA] 
em um projeto SAP S/4HANA.
```

---

## 💡 Dicas de Uso

1. **Para revisão rápida:** Use prompts de Revisão Geral
2. **Para testar conhecimento:** Use prompts de Auto-Avaliação  
3. **Para dúvidas específicas:** Use prompts de Aprofundamento
4. **Para aplicar no trabalho:** Use prompts de Aplicação
